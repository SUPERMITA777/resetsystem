# Reglas del Asistente (Antigravity)

## 1. Idioma
- **Idioma Mandatorio:** Siempre hablaremos en **ESPAÑOL**. Todas las respuestas, explicaciones y comunicaciones con el usuario deben ser en español de ahora en adelante.

## 2. Gestión de Repositorio (Git)
- **Push Automático:** Cada vez que se completen cambios en el código o se finalice una tarea, se debe realizar un commit descriptivo y un `git push` de forma automática, sin necesidad de pedir permiso adicional (a menos que sea un cambio crítico o experimental que requiera validación previa).

## 3. Estética y Diseño (Recordatorio)
- Mantener siempre la estética premium, femenina y elegante solicitada para el proyecto RESET SYSTEM.
